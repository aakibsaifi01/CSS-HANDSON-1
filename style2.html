CSS can be added to HTML document in 3 ways:

  • Inline -by using the style attribute inside HTML elements1
  • Internal -by using a style element in the head section
  • External -by using a link element to link to a external CSS file

When comparing CSS class vs ID ,thhe diffrence is that CSS class applies a style to multipule elements. ID,on the other hand , applies a style to one unique element.ID is also
special in that you can use a special URL to link directly to an element and it's used by Javascript.

In CSS , selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted, a style or set of styles can be applied to
the element

There is a wide range of selectors available. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
